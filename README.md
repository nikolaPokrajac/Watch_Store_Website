# Watch_Store_Website

Watch Store website, created using HTML, SCSS, CSS Grid and Flexbox.

Check out [Live Demo](https://nikolapokrajac.github.io/Watch_Store_Website/)

Inspired by [Stile Theme](https://dribbble.com/shots/5454130-Stile-Theme-Variation/attachments/1180920)

### Getting Started

There are two methods for getting started with this repo.

#### Familiar with Git?

```
> git clone https://github.com/nikolaPokrajac/Watch_Store_Website.git
> cd Watch_Store_Website
> npm install
> npm start or gulp
```

#### Not Familiar with Git?

Click [here](https://github.com/nikolaPokrajac/Watch_Store_Website.git) then click on "Clone or download" and download the .zip file. Extract the contents of the zip file, then open your terminal, change to the project directory, and:

```
> npm install
> npm start or gulp
```
